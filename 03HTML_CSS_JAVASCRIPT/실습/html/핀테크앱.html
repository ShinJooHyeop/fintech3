수수료는 얼마나 되나요?

국내 이체는 기본 무료입니다. 일부 특수 거래는 별도의 수수료가 적용될 수 있습니다.

해외 결제는 지원하나요?

네, 해외 결제를 지원합니다.FTF가 발생할 수 있습니다.
